﻿Basic Commands (Moving Around CLI, Creating Files, Deleting Files, Deleting Directories, Creating Directories)

ls | lists the files that are in a directory
ls –a | shows all files in a directory
ls / | shows the contents of the root directory
pwd | Prints the name of the directory you on in
clear | clears the screen
cd / | Changes directory to root
cd ~ | changes to home directory
touch <file name. extension> | Makes a file in current directory
touch dirname/filename | Makes a new file in a particular directory
nano <file name.extenstion> | Edits a file in nano
ln -s filename linkname | creates link (another file name) to that file
cat <file name. extension> | prints the contents of a file on the screen
mkdir <Directory Name> | Makes a directory
mv <file name> <directory name> | Moves a file to a directory
rm <file name> | removes a file
rmdir <directory name> | removes an empty directory
man <command> | Displays how a command can be used
rm –r | Removed a directory and its contents
rm –rf / | Deletes everything under root (kills the system)﻿
apropos <word or phrase> | get a list of commands relating to that word or phrase
hardinfo | Get summary of hardware specs
# | bash comment
head <file or directory> | prints first 10 lines of that file
tail <file or directory> | prints last 10 lines of that file/directory
tail -f | follow file input
shutdown + extension:
-r | restart
-h | halt/turn off
+ now/+(number of minutes)
shutdown -r +10 | restart in 10 minutes
poweroff | shutdown computer




Shortcuts:
ctrl + a | start of current line
ctrl + e | end of current line
ctrl + r | reverse history search
ctrl + c | interrupts current function (cancel)
